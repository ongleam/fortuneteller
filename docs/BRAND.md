# 점순이 브랜드 아이덴티티

> **이 문서는 점순이 서비스의 브랜드·UI 아이덴티티에 대한 SSOT(Single Source Of Truth)입니다.**
>
> 랜딩·소개팅 화면(`/`, `/discover`, `/matches`, `/profile/edit`)의 색·글꼴·모티프·말투는 이 문서를 따릅니다.
> 코드에서 쓰는 토큰은 [`apps/web/src/components/brand/theme.ts`](../apps/web/src/components/brand/theme.ts)에 모아 두었고, 공용 요소는 [`elements.tsx`](../apps/web/src/components/brand/elements.tsx)에 있습니다.
>
> **⚠️ 브랜드가 바뀔 때(색·글꼴·모티프·말투)마다 이 문서와 `theme.ts`를 함께 업데이트하세요.**

## 한마디

**점순이 — 사주로 인연을 잇다.**
생년월일로 궁합을 보고, 잘 맞는 사람을 먼저 만나는 사주 궁합 소개팅.

## 톤 & 무드

- **조선 시대 사주 명식(命式) 문서**의 분위기: 오래된 **한지(양피지)** 위에 **먹(墨)**으로 쓴 붓글씨.
- 신비롭고 정갈하며 따뜻함. 요란하지 않고 여백이 있음.
- 화려한 그라디언트·네온 대신, 종이 질감·붉은 낙관·얇은 괘선으로 절제된 품위를 준다.

## 색 (Palette)

| 이름          | 값                    | 쓰임                        |
| ------------- | --------------------- | --------------------------- |
| 먹 (ink)      | `#241c12`             | 제목·본문 기본 글자         |
| 옅은 먹 (sub) | `#6b5638`             | 보조 문구·설명              |
| 낙관 (seal)   | `#9c2b1f`             | 강조·주요 버튼·도장         |
| 금니 (gold)   | `#a8842c`             | 얇은 괘선·테두리            |
| 한지 (paper)  | `#ebe0c8`             | 페이지 바탕                 |
| 한지 카드     | `#f4ecd8`             | 카드·입력창 바탕(살짝 밝게) |
| 괘선 (line)   | `rgba(60,40,20,0.28)` | 구분선·테두리               |

**오행(五行) 색** — 사주 요소를 나타내는 상자 테두리 색:
목(木) `#2b4a7a` · 화(火) `#9c2b1f` · 토(土) `#a8842c` · 금(金) `#8a8a86` · 수(水) `#2b2b2b`

바탕은 단색이 아니라 은은한 얼룩(radial-gradient)과 가장자리 그을림(vignette)을 얹어 한지 질감을 낸다.

## 글꼴 (Typography)

| 용도                        | 글꼴                             | 비고                        |
| --------------------------- | -------------------------------- | --------------------------- |
| 브랜드·큰 제목 (디스플레이) | **Nanum Brush Script** (붓글씨)  | 손으로 쓴 붓글씨 느낌       |
| 제목·본문                   | **Nanum Myeongjo** (명조/세리프) | 전통적이고 읽기 좋은 세리프 |

- 웹폰트는 Google Fonts로 로드(`FontLinks`), 유니코드 범위로 한글만 필요 시 내려받는다.
- 굵은 제목은 명조 800, 본문은 명조 400.

## 모티프

- **낙관(도장)** — 붉은 사각 테두리에 한자(`占`·`緣`·`命`)를 넣어 살짝 기울인 도장. 브랜드 마크·섹션 앵커로 사용.
- **붓 밑줄** — 제목 아래 거칠게 그은 붓 자국(SVG). 깔끔한 직선 대신 손맛.
- **오행 타일** — `木火土金水`를 색 상자에 담은 장식 띠(사주 명식 인용).
- **한자 소제목** — 섹션마다 한글 위에 한자 라벨: 궁합=`宮合`, 인연=`因緣`, 명식=`命式`, 사주=`四柱`.
- **한자 숫자** — 순서는 `一 二 三`.
- 버튼은 **각진 사각형**(둥근 모서리 지양)에 낙관 붉은색 또는 먹색.

## 말투 (Voice)

- **쉽고 짧고 편안하게. 핵심만.** 어려운 한자어·전문 용어·비문(어색한 문장) 금지.
- 존댓말, 담백하게. 예:
  - "나와 잘 맞는 사람부터 보여드려요."
  - "둘 다 좋아하면 인연이 돼요."
  - "생년월일만 있으면 충분해요."
- 한자는 **장식(소제목·도장)** 으로만 쓰고, 읽어야 하는 문장은 늘 쉬운 한글로 쓴다.

## 코드에서 쓰기

```ts
import { brandColors, brandFont, OHAENG } from "@/components/brand/theme";
import {
  ParchmentShell,
  BrandMark,
  Seal,
  BrushRule,
  SectionTitle,
  OhaengRow,
} from "@/components/brand/elements";
```

- 페이지는 `<ParchmentShell>` 로 감싼다(한지 바탕·그을림·폰트·폰트링크 포함).
- 브랜드 로고는 `<BrandMark/>`, 도장은 `<Seal char="緣"/>`, 제목은 `<SectionTitle>`.
- 색·글꼴은 하드코딩하지 말고 `brandColors`·`brandFont` 를 쓴다.
